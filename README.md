# Celebal Technologies - Data Engineering Internship

## Week 1 Assignment: Basic Data Exploration and Cleaning using Pandas

### Objective
Perform exploratory data analysis (EDA), data cleaning, and derive meaningful insights from an e-commerce shopping dataset.

### Dataset
Combined_dataset.csv - Contains product-level information including price, rating, category, and customer reviews.

### Steps Performed
1. Loaded the dataset into a Pandas DataFrame
2. Explored data structure (shape, columns, data types)
3. Cleaned data - handled missing values, removed duplicates, converted price columns to numeric
4. Feature engineering - created price_difference and popularity_score columns
5. Performed Univariate, Bivariate, and Category-level analysis
6. Created visualizations (histograms, bar charts, boxplots)
7. Derived key business insights from the analysis

### Files
- `Week1_Assignment.ipynb` - Main notebook with code and analysis
- `Combined_dataset.csv` - Original dataset
- `cleaned_shopping_dataset.csv` - Cleaned output dataset

### Key Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook
